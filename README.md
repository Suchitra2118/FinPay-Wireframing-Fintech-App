## Project Name
Wireframing for a Fintech App

## Project Description
This project focuses on designing low-fidelity wireframes for a fintech mobile application called FinPay. The wireframes demonstrate the user journey for authentication, money transfer, transaction tracking, profile management, notifications, and settings.

## Tools Used
- Figma

## Screens Included
1. Login Screen
2. Dashboard Screen
3. Send Money Screen
4. Transaction History Screen
5. Profile Screen
6. Receive Money Screen
7. Success Screen
8. Alerts Screen
9. Settings Screen

## Outcome
Designed a complete user flow for a fintech application using wireframes and UX principles.
